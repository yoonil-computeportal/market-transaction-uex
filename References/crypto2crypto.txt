Crypto to Crypto
POST
https://uexswap.com/api/partners/swap/initiate-crypto-to-crypto
Initiates a new cryptocurrency-to-cryptocurrency swap.

Validates all provided inputs before creating the order.

Note: Before using this API, please review the usage guidelines and instructions at: https://uex-us.stoplight.io/docs/uex-us/branches/main/4qv65b3k0kxfm-uex-us-swap
Request

Body



application/json

Creates a new cryptocurrency-to-cryptocurrency swap order. The system validates all inputs and, if valid, returns details including the deposit address where the base currency should be sent.
send_amount
number<float>
required
Amount of the base cryptocurrency to swap.
Example:
1
from_currency
string
required
Symbol/code of the source cryptocurrency (base currency) (e.g., "BTC").
Example:
BTC
base_currency_chain_id
string
required
Blockchain network name of the base currency (e.g., "BTC" or network-specific code).
Example:
BTC
to_currency
string
required
Symbol/code of the target cryptocurrency to receive (quote currency).
Example:
USDT
quote_currency_chain_id
string
required
Blockchain network name of the quote currency.
Example:
TRX
userWallet
string
required
The wallet address where the swapped funds (quote currency) should be sent.
Example:
TQ3LqkLj4FVq7ZQf8uqJHfEgF7rHjsq2KE
receive_tag
string or null
Optional memo/tag for destination (used by some blockchains like XRP, XLM).
Example:
null

extend
object
required
Additional metadata.
ref_code
string
required

5drfo01pgq88
